# Overview
**ofxSurfingOsc** is a predefined Osc Manager as a template of a combo for Master / Slave apps.

## Screenshot
![](Capture.PNG)

## Features
- Predefined common template.
- 8 bool bangs, 8 bool toggles, 8 float values, 8 int numbers.
- Internal params with GUI.
- Persistent settings.
- Plots.
